The Homer Multitext project on github

Archival data and software for working with editions of Homeric manuscripts

Current

  • hmt-archive: the project's central data archive
  • hmt-digital: digital services for working with the Homer Multitext project archive
  • hmt-vm: a Vagrant virtual machine definition for editorial work on the Homer Multitext project
  • hmt-mom: Mandatory On-going Maintenance, a build project for testing and validating editorial work on the Homer Multitext project
  • hmt-utils: a JVM library implementing some HMT-specific functionality, such as tokenization of texts following HMT project conventions

In progress

  • morpheus: a fork of the Alpheios modification of the Perseus morphological parser, extending the morphological data set to cover all vocabulary in the Homer Multitext project archive
  • hmt-editors-guide: manual for HMT editors
