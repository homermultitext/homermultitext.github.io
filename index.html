Explore the contents of the HMT archive

  • alpha-search: simple alphabetic search of Greek texts in the HMT archive. Filter by manuscript and/or text.
  • codex-browser: visual browser of photographed and documented codices in the HMT archive.
  • lightbox: paged browsing of image collections.

Reference lists

  • authlists: searchable tables of authority lists.

The HMT archive

The HMT project's work editing manuscripts of the Iliad with scholia is managed in this github repository.

The current published release of the hmt-archive is always available as a plain-text file in the Cite EXchange format from this link.
